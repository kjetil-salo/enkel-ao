# Brukervurdering av Enkel-AO

*Vurdering utført 23. januar 2026*

## Styrker
- **Fokusert**: Gjør én ting godt - rask registrering i felt
- **Offline-først**: Kritisk for feltbruk uten dekning
- **Null friksjon**: Ingen innlogging, åpne og bruk
- **Mobiltilpasset**: Touch-vennlige pills, god størrelse
- **iPad-layout**: Bra for trekktelling med oversikt

## Svakheter
- **Ingen synk**: Data kun på én enhet/nettleser
- **Sårbar lagring**: localStorage kan slettes ved uhell
- **Ingen onboarding**: Nye brukere må finne ut selv
- **AO-avhengig**: Import krever manuell liming

## Forslag til fremtidige forbedringer
1. **Eksport til fil** (JSON/backup) - sikkerhetskopi
2. **Hurtigtaster for antall** (1-9 direkte registrering)
3. **Siste brukte arter** øverst i søk
4. **Visuell bekreftelse** når obs lagres (kort feedback)

## Konklusjon
Solid app for målgruppen (fuglekikkere i felt). Enkel, rask, fungerer offline. Største risiko er datatap ved nettleser-reset. Vurder eksport/backup-funksjon.
