# Fugleobservasjoner v1.13.0

**Utgitt:** 27. januar 2026

## 🎨 UI/UX forbedringer

### Kompaktere søkefelt
- Strammere padding i søkefelt og resultatliste
- Større artsnavn (1.25rem i søkefelt, 1.08rem i resultater)
- Mindre border-radius (6px) for moderne look

### Antall-felt forbedret
- Aksepterer kun sifre (blokkerer bokstaver, e, ., +, -)
- `inputmode="numeric"` for mobil
- `pattern="[0-9]*"` for validering
- Bedre brukeropplevelse ved registrering

### Layout-optimalisering
- Forbedret spacing mellom elementer
- Bedre visuell hierarki
- Kompaktere iPad-layout med mindre vertikal padding

## 📱 Mobilforbedringer
- Numerisk tastatur på mobil (kun sifre)
- Bedre touch-targets
- Optimalisert for feltbruk

## Teknisk
- HTML input-attributter oppdatert
- CSS-justeringer for spacing
- Forbedret responsiv design
